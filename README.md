# Pult: Удаленное управление сервером

Приложение для удаленного мониторинга системы и выполнения команд.

Проект состоит из бэкенда на Python (FastAPI) и нативного Android-клиента.

https://shaperones0.github.io/pult/

## Структура

- `pult-server/`: Демонстрационный бэкенд, написанный на FastAPI.
- `pult-client-android/`: Нативное Android-приложение, написанное на Kotlin.

## Фичи

- Стриминг статистики использования CPU и RAM с сервера на клиент через WebSockets (`/ws/metrics`).
- Запуск терминальных (или кастомных текстовых, в зависимости от реализации на сервере) команд через HTTP (`/action`) с трансляцией логов выполнения.
- Защита ручек с помощью обязательного заголовка `X-Api-Key`.
- Android-клиент:
    - Быстрый вход по URL и API-ключу или через сканирование QR-кода.
    - Интерактивные графики для визуализации нагрузки на систему.
    - Локальная база данных для сохранения избранных команд и отслеживания истории выполнения.
    - Фоновый воркер для постоянного мониторинга.

## Технологи

- Бэкенд: Python 3.14, `uv`, FastAPI, WebSockets.
- Клиент: Kotlin, Android SDK, Retrofit 2 & OkHttp, MPAndroidChart, ZXing Android Embedded.

## Запуск сервера (`pult-server`)

```bash
cd pult-server
uv sync
uv run uvicorn app:app --host 127.0.0.1 --port 8000
```

Интерактивная документация API (Swagger UI) доступна по адресу `http://localhost:8000/docs`.

## Запуск клиента (`pult-client-android`)

1. Откройте директорию `pult-client-android` в Android Studio.
2. Дождитесь синхронизации Gradle-проекта.
3. Соберите и запустите приложение на эмуляторе или физическом устройстве (требуется Android 8.0 / API 26 или выше).
